# my-daily-agent

A standalone local opencode workspace configuration for a daily development agent.

## Structure

- `.opencode/opencode.json` — workspace-local opencode config
- `.opencode/agents/my-daily-agent.md` — standalone daily agent definition

## Usage

Open this folder as an opencode workspace:

`C:\Users\Administrator\Desktop\Agent\my-daily-agent`

The workspace-local config will default to the `my-daily-agent` agent.
